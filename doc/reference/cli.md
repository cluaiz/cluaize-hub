# CLI reference

The `cluaiz` CLI manages skill and plugin installation, updates, and discovery.

## Skills

### Search

```bash
cluaiz skill search <query>
```

Search the registry for skills matching a keyword or description.

### Install

```bash
cluaiz skill install <name>
```

Downloads the skill folder (SKILL.md + all linked assets) to
`~/.cluaiz/skills/<name>/`.

The CLI resolves the skill by checking:
1. Exact ID match (e.g., `cluaiz.skill.ops.storage`)
2. Name match (e.g., `storage-probe`)
3. Folder path match (e.g., `Independent-ops/storage-probe`)

### Update

```bash
cluaiz skill update <name>
cluaiz skill update --all
```

Re-downloads the latest version from the registry.

### Remove

```bash
cluaiz skill remove <name>
```

Removes the local installation from `~/.cluaiz/skills/<name>/`.

### List

```bash
cluaiz skill list
```

Lists all locally installed skills with their versions.

## Plugins

### Install

```bash
cluaiz plugin install <name>
```

Downloads the plugin folder to `~/.cluaiz/plugins/<name>/`.

### Remove

```bash
cluaiz plugin remove <name>
```

## Registry

The CLI fetches the skill index from the `registry.json` hosted on the
`cluaiz-skills` GitHub repository. The registry is auto-generated by the
CI pipeline whenever a skill is added or updated.

## Environment

| Variable | Effect |
|---|---|
| `CLUAIZ_HOME` | Override the default install directory (default: `~/.cluaiz`). |
| `CLUAIZ_REGISTRY_URL` | Override the registry URL for custom or private registries. |
